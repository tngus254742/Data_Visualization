# Data_Visualization
## Explanatory Data Analysis with various visualizations

In this project, you can have a look on some explanatory data analysis of three datasets, "seeds.txt", "boston.txt" and "mpg.txt" in R.
The detailed information of this dataset can be found at:
[https://archive.ics.uci.edu/ml/datasets/seeds][1]
[https://www.kaggle.com/c/boston-housing.][2]
[https://www.rdocumentation.org/packages/ggplot2/versions/3.3.3/topics/mpg.][3]


### ggplot2

ggplot2 is a data visualization package for the statistical programming language R.

#### Installation in R

```
install.packages("ggplot2")
```

```
library(knitr)
```
